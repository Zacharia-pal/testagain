Games & Personal Projects

Scope: Creative development and side projects
Status: Ongoing

Beyond the Enterprise

Professional work tells one part of the story. Personal projects reveal the curiosity and drive that fuel everything else. Outside of client engagements and enterprise deliverables, there is a continuous thread of creative development and experimentation that keeps technical skills sharp and explores areas that traditional projects do not always reach. These side projects are where new ideas get tested, new technologies get explored, and the boundaries of what is possible get pushed in a low-stakes, high-creativity environment.

Game Development

Game development serves as a creative outlet that combines programming, design, logic, and storytelling into a single discipline. Building games requires the same structured thinking and problem-solving skills used in enterprise software development, but applies them to interactive experiences where user engagement is the primary measure of success. From designing game mechanics and writing logic to handling rendering and user input, game development exercises a broad range of technical skills in an environment where experimentation is encouraged and iteration is fast.

The AZ Khonnect Website

This website is itself a personal project. AZ Khonnect was designed and built from scratch as a professional portfolio and business platform. It demonstrates front-end development skills including responsive design, clean HTML and CSS architecture, JavaScript interactivity, and attention to user experience. The site serves as both a showcase for completed work and a living project that continues to evolve with new features and content.

Why It Matters

Personal projects are where passion meets practice. They demonstrate a genuine interest in technology that goes beyond the requirements of any single role or client engagement. The skills developed through game design, web development, and creative coding feed directly back into professional work, bringing fresh perspectives and a broader technical vocabulary to every project. A well-rounded developer is one who builds not only for clients but also for the joy of building itself.
